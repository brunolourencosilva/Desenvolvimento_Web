1. Display: Block

Elementos com display: block ocupam toda a largura disponivel,criando uma nova linha antes e depois do elemento.

Este é um bloco,ele ocupa toda a largura disponivel e cria uma quebra de linha.
Outro bloco,o que mostra que os blocos vão para a proxima linha.

2. Diplay: Inline

Elementos com diplay: inline se alinham horizontalmente,respeitando apenas o seu conteudo e não criam quebras de linha.

Este é um elemento inline. Outro elemento inline. Mais um elemento inline.

3. Display: Inline-block

Elementos com display: inline-block se alinham horizontalmente, mas podem ter largura e altura definidas,ao contrario do inline.

Este é um elemneto inline-block.
Outro elemento inline-block.
Mais um elemento inline-block.
